# Project Title

This is a static website project with English and Arabic language support.

## Description

The website consists of several pages including index, services, team, references, and contact. It has both English and Arabic versions located in the root directory and the `ar/` subdirectory respectively.

## Structure

- `index.html` - Main page (English)
- `services.html` - Services page (English)
- `team.html` - Team page (English)
- `references.html` - References page (English)
- `contact.html` - Contact page (English)
- `ar/` - Arabic version directory
  - `index.html` - Main page (Arabic)
  - `services.html` - Services page (Arabic)
  - `team.html` - Team page (Arabic)
  - `references.html` - References page (Arabic)
  - `contact.html` - Contact page (Arabic)
- `google35a042ae70cbe09f.html` - Google verification file

## How to Run

Simply open `index.html` in your web browser to view the English version of the website. For the Arabic version, open `ar/index.html`.
